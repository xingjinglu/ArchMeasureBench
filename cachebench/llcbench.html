To get LLCbench:

Download the distribution: llcbench.tar.gz.

Or access the CVS repository directly:

$ setenv CVSROOT :pserver:anonymous@hera.cs.utk.edu:/cvs/homes/llcbench
$ cvs login
Password: <cr>
$ cvs checkout llcbench
$ cvs logout

Or surf the CVS tree with your browser

To use LLCbench:

UNIX> gzip -dc llcbench.tar.gz | tar xvf -
UNIX> cd llcbench
UNIX> make

Please use one of the following targets:

        alpha
        alpha-mpich
        linux-lam
        linux-mpich
        o2000
        pow2
        pow3
        ppc
        solaris
        solaris-mpich
        t3e
        reconfig (to bring this menu up again)

Choose the appropriate target: 

UNIX> make solaris-mpich
ln -s conf/sys.solaris sys.def

Optionally edit/review the compile options:

UNIX> vi sys.def

Build the benchmarks: 

UNIX> make
Please use one of the following targets:

For all three : compile, run, graph, clean, clobber, reconfig
For BlasBench : blas-bench, blas-run, blas-graph
For CacheBench: cache-bench, cache-run, cache-graph
For MPBench   : mp-bench, mp-run, mp-graph
UNIX> make compile

Optionally edit/review the run-time options:

UNIX> vi user.def

Run the benchmarks:

UNIX> make run

Examine and graph the results:

UNIX> cd results
UNIX> ls *.dat *.tar
UNIX> make graph
UNIX> ls *.ps
UNIX> ghostview *.ps
